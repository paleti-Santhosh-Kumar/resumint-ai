# RESUMINT AI 🚀

The Intelligent AI Career Operating System.

Build, score, tailor, and generate recruiter-ready resumes with AI.

## Features

* AI Resume Builder
* Resume Templates
* Resume Scoring
* Live Resume Editor
* Mock Interview Preparation
* Export Resume
* Modern Responsive UI
* Fast Vite + React Experience

## Tech Stack

* React
* Vite
* TypeScript
* Tailwind CSS
* Supabase

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Deploy

```bash
npm run deploy
```
